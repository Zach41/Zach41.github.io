Rust

Rust是一种注重高效、安全、并行的系统程序语言。它有以下特定：

  • 零开销抽象
  • 转移语义
  • 保证内存安全
  • 没有数据竞争的线程
  • trait泛型
  • 模式匹配
  • 类型推断
  • 极小的运行时
  • 高效的C绑定

Rust是一门有希望代替C++的语言，所以确实值得学习。于是这也就鼓捣了以下emacs下的环境配置。
